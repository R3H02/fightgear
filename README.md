FightGear - Loja Virtual de Produtos de Luta

Projeto de e-commerce desenvolvido em PHP e MySQL, com foco em gerenciamento de produtos e controle de estoque.

Funcionalidades:

Sistema de autenticação com níveis de acesso (cliente e administrador)
CRUD completo para gerenciamento de produtos (criação, leitura, atualização e exclusão)
Painel administrativo para controle de dados
Interface responsiva para desktop e dispositivos móveis

Tecnologias utilizadas:

PHP
MySQL (phpMyAdmin)
HTML, CSS e JavaScript

Diferenciais:

Estrutura organizada e de fácil manutenção
Validação de dados e uso de boas práticas de segurança
Proteção contra SQL Injection com prepared statements
